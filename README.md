# Internship Task - 2: NLP using PyTorch

## Description
This project builds a simple text classification model using PyTorch and TorchText. It classifies news into 4 categories using AG_NEWS dataset.

## Setup
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the project:
   ```
   python text_classification.py
   ```

3. Outputs:
   - Trained model: `model/text_model.pth`
   - Training graph: `outputs/loss_accuracy_plot.png`
